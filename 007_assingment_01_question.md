Objective:
Practice using variables, arithmetic operations, and string formatting in Python.

-------------------------------------------------
Part 1: Variables
-------------------------------------------------

Question 1: Rectangle Area

Write a Python program that:

- Creates two variables:
  length = 8
  width = 12

- Calculates the area of the rectangle.

- Prints the result.

-------------------------------------------------

Question 2: Circle Area

Write a Python program that:

- Creates two variables:
  pi = 3.14
  radius = 7

- Calculates the area of the circle using the formula:

  Area = π × r²

- Prints the result.
